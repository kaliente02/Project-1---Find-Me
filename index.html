How to use Find Me

Users can use FindMe by first signing in or creating an account using their school email and student information. After logging in, they can report a lost item by filling in the item name, description, category, and the location where it was last seen. If a user finds an item on campus, they can also report it by submitting the item details and uploading a photo to help identify it. Users may search for items through the search bar or browse the list of recently reported items to check if their belongings have been found. If a user recognizes an item as theirs, they can open the item details and submit a claim request so the administrator can verify and return the item to the rightful owner.
